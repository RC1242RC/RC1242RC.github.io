## Detailed Description


**Author**: Anders Kvellestad ([a.kvellestad@imperial.ac.uk](mailto:a.kvellestad@imperial.ac.uk)) 

**Date**: 2019 Sep

A big multimap connecting Pythia process codes to the PID numbers for the two final state particles.

This is needed to use external cross-section calculators for weighting Pythia events.
